By Paulo Alem

tags: pythoncódigo idiomáticopythonicopythonic

Lá na Quatix, onde trabalho, rolam Lightning Talks nas terças-feiras. Estou trabalhando lá a pouco mais de um ano e, de alguma maneira, escapei de apresentar um até a terça passada(falar em público nao é lá minha atividade preferida (: ).

O talk, "Python do Além", foi sobre código idiomático, mais especificamente python idiomático (aka pythônico).

Para criar a apresentação usei Landslide, uma biblioteca que gera slides html5 e pdfs a partir de reStructuredText. Vou disponibilizar os slides dessa apresentação(e todas as outras que eu fizer nesse formato) nesse repositorio no github.
